## 개발 환경
1. IDE : IntelliJ IDEA
2. Spring Boot 3.3.6
3. JDK 17
4. mysql
5. Spring Data JPA
6. Thymeleaf

## 기본 기능 (강의 제공 기능)
1. 글쓰기(/board/save)
2. 글목록(/board/)
3. 글조회(/board/{id})
4. 글수정(/board/update/{id})
5. 글삭제(/board/delete/{id})
6. 페이징처리(/board/paging)
7. 파일첨부
8. 댓글
